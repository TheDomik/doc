![[Pasted image 20210918152655.png]]

Use this node to filter random cells  
	
**Inputs:**

- **In** - cells to filter

**Outputs:**

- **Success** - filtered cells
- **Failed** - —Åells that have not been filtered 

**Properties:**

- **Count** - how many cells you want to filter
