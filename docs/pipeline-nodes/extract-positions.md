# **Extract Positions** 

![[Pasted image 20211009213733.png]]

This node extract positions of input cells to use them later. Pretty useful to save positions of some processed cells.  
	
**Inputs:**

- **Cells** - cells to extract

**Outputs:**

- **Positions** - extracted positions
