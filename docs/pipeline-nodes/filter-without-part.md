# **Filter Without Part**

![[Pasted image 20210922162138.png]]

This node allows you to filter cells without some part.  

**Inputs:**

- **In** - cells to filter

**Outputs:**

- **Out** - cells without a target Part

**Properties:**

- **Part** - part what you want to use as exclude filter
