# **Filter By Positions**

![[Pasted image 20210918173720.png]]

This node filters cells by positions.  
	
**Inputs:**

- **In** - cells to filter
- **Positions** - Positions of cells which should be filtered

**Outputs:**

- **Out** - filtered cells
