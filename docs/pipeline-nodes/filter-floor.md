# **Filter Floor**

![[Pasted image 20210918154450.png]]

Use this node to filter concrete floor.  
	
**Inputs:**

- **In** - cells to filter
- **Floor** - input number (optional).

**Outputs:**

- **Out** - filtered cells

**Properties:**

- **Number** - Number of the floor to filter
