# **Exclude** 

![[Pasted image 20210923143504.png]]

This node excludes cells B from cells A.   
	
**Inputs:**

- **A** - cells which you want to process
- **B** - cells which you want to exclude from **A**

**Outputs:**

- **Out** - result
